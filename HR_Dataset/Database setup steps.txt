Database Setup (Source & Target Data)
we need to load the data that we will use during our course, and to do so follow the next steps:

1- Open SQL Server Management Studio 

2- Connect to the Server using the credentials as we described in the previous lecture

3- Click on “New Query”

4- Write the following query to create the database, then click on Execute

CREATE DATABASE HR;

5- Open the file attached to this lesson named ‘Table Creation HR’, copy its content, and paste it into a new query to create the tables

6- Make sure that the database and tables have been created successfully 

7- Open the file attached to this lesson named ‘Load HR data’, copy its content, and paste it into a new query to create the tables

8- Now database, tables, and data are ready, let’s dive into the first section of our course.
